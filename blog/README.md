# 📝 Django Blog App

A full-featured blog platform built with Python and Django.

---

## 🚀 Features

- User login and logout
- Create, edit, and delete blog posts
- Upload cover images for posts
- Comment on posts and delete your own comments
- Pagination on home page
- Django admin panel
- Clean responsive UI with Bootstrap 5

---

## 🛠️ Tech Stack

- **Backend:** Python, Django
- **Frontend:** HTML5, CSS3, Bootstrap 5
- **Database:** SQLite3

---

## ⚙️ Setup
```bash
git clone https://github.com/DSBaibhav/blog-app.git
cd blog-app
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python manage.py migrate
python manage.py createsuperuser
python manage.py runserver
```

Open at `http://127.0.0.1:8000`
